## 프로젝트 소개

좋아하는 가수의 콘서트를 함께 갈 사람, 이번주 주말에 맛집 투어를 함께할 사람 등 하루 단위의 간단한 모임을 함께할 인연을 모집하는 서비스를 제공합니다. 

주최자가 모임의 카테고리, 희망 인원과 장소, 금액 등을 작성하여 게시하면, 참여 신청하여 모임에 함께 합니다.
